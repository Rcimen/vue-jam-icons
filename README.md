# vue-jam-icons
VueJS component for jam icons v2 https://jam-icons.com

## Get Started

Install via npm

```
$ npm install vue-jam-icons
```

## Usage

``` JavaScript
// main.js
import Vue from 'vue'
import VueJamIcons from 'vue-jam-icon'

Vue.use(VueJamIcons)
```

``` JSX
<template>
  <jam-backpack/>
</template>
```
