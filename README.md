# vue-jam-icons
VueJS component for jam icons v2 https://jam-icons.com

## Get Started

Install via npm

```
$ npm install vue-jam-icons
```

## Usage
Register vue-jam-icons globally:
``` JavaScript
// main.js
import Vue from 'vue'
import VueJamIcons from 'vue-jam-icon'

Vue.use(VueJamIcons)
```
Use any icon prefixed by `jam-` in a self-closing tag:
``` JSX
//MyComponent.vue
<template>
  <jam-backpack/>
</template>
```
